# SciLens — Medical Device Scoring Guide

## Scope
Diagnostic devices, implants, surgical tools, IVD, wearables, digital health, ISO 13485.

## TRL Calibration
| Paper evidence | TRL |
|---------------|-----|
| Device concept, computational model | 1–2 |
| Bench testing, prototype | 3–4 |
| Biocompatibility + functional testing | 5 |
| Clinical feasibility study | 6 |
| Pivotal clinical study | 7–8 |
| Cleared/approved device, post-market | 9 |

## Evidence Quality Priority
Clinical study (patient outcomes) > bench + biocompatibility > computational/simulation

## Key Analysis Focus
- FDA classification: Class I/II/III → 510(k) / De Novo / PMA pathway
- CE Mark under MDR (EU): Clinical Evaluation Report, Notified Body involvement
- ISO 13485 (QMS), IEC 60601 (electrical safety), IEC 62304 (software)
- Biocompatibility per ISO 10993
- For software: IEC 62304 lifecycle requirements

## Market Context Instructions
Device approvals are faster than drugs. Use trial/approval data to indicate
competitive density in the device category. FDA clearance (510(k)) is 6–18 months;
PMA is 18–36 months — reference this in timeline assessment.

## Risk Factors to Prioritize
1. Predicate device availability for 510(k) path
2. Clinical evidence requirements per device class
3. Cybersecurity risk for connected/digital health devices
4. Post-market surveillance obligations (MDR/IVDR)
5. Reimbursement pathway (CPT coding, coverage decisions)
