# SciLens — Pharma Clinical Scoring Guide

## Scope
Drug development, clinical trials, pharmacology, therapeutics, FDA/EMA approval pathways.

## TRL Calibration (based on THIS paper's own evidence)
| Paper evidence | TRL |
|---------------|-----|
| Hypothesis / computational modeling only | 1 |
| In silico validation, mechanism proposed | 2 |
| In vitro proof of concept | 3 |
| In vivo (animal models), PK/PD data | 4 |
| Phase I data (safety, first-in-human) | 5 |
| Phase II data (efficacy, dose-ranging) | 6 |
| Phase III data (pivotal, large-scale) | 7–8 |
| Approved / post-marketing evidence | 9 |

## Evidence Quality Priority
meta_analysis > rct > cohort > case_control > in_vivo > in_vitro > in_silico > review

## Key Analysis Focus
- Mechanism of action clarity and novelty
- Safety signals and toxicity profile
- Unmet medical need addressed
- Differentiation vs. existing approved therapies
- Realistic path to IND → NDA/BLA
- Regulatory signals: orphan drug, breakthrough therapy, fast track

## Market Context Instructions
Real trial and approval data has been provided. Use it to:
- Describe whether the space is saturated or open
- Assess if this approach is first-in-class or fast-follower
- Reference the number of active trials and approvals as context — do NOT invent dollar figures

## Risk Factors to Prioritize
1. Clinical translation failure rates for this target class
2. Safety precedent in the target space
3. Regulatory path complexity (NDA vs. BLA, novel vs. precedented)
4. Manufacturing complexity for the proposed modality
5. IP landscape signals from the paper's novelty claims
